## About Laravel MQ

This application uses a message broker to communicate with two services. The User service and Notification service. The message broker used is RabbitMQ.


## Setup Laravel MQ

Run docker compose up in your terminal to run the application.
Please ensure to have dicker Desktop installed your local machine.

## End points

POST - http://127.0.0.1/api/create-user
Data - {"first_name","last_name","email"}
